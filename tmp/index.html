node.js

What is node.js

Is a platform that allows us to run JS on a computer or a server, this way is not completely necesary to learn a different programming language to crate a whole app.

With node.js we can read, delete and updoate files like any other server side language.

With node.js we can communicate easily with a database

Why node.js

It uses JS

Is very fast, it runs on the V8 engine and uses nonblocking code

Huge ecosystem of open source packages (npm)

Great for real time services like chats

The V8 engine

The engine converts JS to machine code which is understandable by the computer.

node.js is written in c++ and V8 as well, V8 is embedded in node.js. V8 gives the powers to JS to do things that JS was not capable to do before like connecting with databases.

The Global Object

Cuando estamos en un explorador, el objeto global es window y podemos usar events que se encuentran viviendo en este entorno console.log alert o setTimeout, en node.js el objecto global ya no es window porque no estamos en un explorador, en node el objecto global se denomina global y aquí como en el explorados también tenemos acceso a algunos métodos que podemos usar de inmediato.
